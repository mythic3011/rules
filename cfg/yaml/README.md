## 远程 YAML 示例

这里存放本项目的远程 YAML 示例文件。

- `Custom_Clash.yaml`：标准模板
- `Custom_Clash_GFW.yaml`：GFW 场景模板
- `Custom_Clash_Lite.yaml`：精简模板
- `Custom_Clash_Full.yaml`：完整模板
- `Custom_Clash_AI.yaml`：AI 专用模板（多 Provider 拆分，面向非 GFW 用户）
- `Custom_Clash_AI_Strict.yaml`：AI 专用严格模板（默认仅走代理，按需仅放行 Copilot/Perplexity/Gemini/Grok/Poe 直连）
