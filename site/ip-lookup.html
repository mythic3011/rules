IP lookup (CIDR assets only)

This first version only searches CIDR-capable assets from the repository inventory. It does not search domain-only assets, mixed classical assets, or unsupported binary assets.

Current scope: IPv4 only, exact IP and CIDR containment only. `mixed` assets are intentionally skipped and reported as skipped, not silently ignored.
Audit trail: rule asset inventory and IP match schema.
